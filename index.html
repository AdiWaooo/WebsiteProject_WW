    Project Presentation

    Purpose: This interactive web application explores the history of the World Wars through a dynamic, visual, and highly accessible interface.

    Design & UX: Designed with a premium dark-mode aesthetic utilizing 'Cinzel' for cinematic headings and 'Inter' for highly readable body text. The layout uses CSS Grid and Flexbox for complete responsiveness.

    Innovative Features:
    • Revolving interactive timeline carousels.
    • Full-screen theatre exploration with categorized history.
    • Hover-activated Lightbox zoom for historical imagery.
    • Auto-playing image slideshows for multi-image figures.
    • Seamless, JavaScript-driven single-page navigation without page reloads.

    Code Quality: Built with clean, semantic HTML5, modern CSS3 variables for dynamic theming, and modular JavaScript.
